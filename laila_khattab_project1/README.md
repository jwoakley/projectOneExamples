
# Is there anybody in there?

# project overview

This is “Is there anybody in there?” by Laila Khattab. It is a personal website for a school project meant to visualize personal data

# technical overview 

The site is coded in html and css that features data samples on aspects of my life. There are three pages that link to each other and one css style sheet that applies to the whole site.
